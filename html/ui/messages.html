Tooltips

The tooltip plugin generates content and markup on demand. Options can be passed via data attributes or JavaScript.

Trigger the tooltip by simply specifying data-toggle="tooltip" on an element.

Popovers

Add small overlays of content, like those on the iPad, to any element for housing secondary information.

Trigger the popover by simply specifying data-toggle="popover" on an element.
